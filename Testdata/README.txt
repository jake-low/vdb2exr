Contains Content from or based on Walt Disney Animation Studios Cloud Data Set
https://www.disneyanimation.com/data-sets/?drawer=/resources/clouds/

PACKAGE CONTENTS:

wdas_cloud.vdb : a heterogeneous volumetric cloud model containing density data
wdas_cloud_*.vdb : lower-resolution versions at half, quarter, eighth, and sixteenth resolution
wdas_cloud_*.exr : tiled textures create from vdb files with this tool.
